# 预算看板开发文档

## 1. 模块目标

预算看板用于在微信小程序内按自然月管理预算，核心目标是：

- 直观展示每个预算项的已使用进度（多彩进度条）
- 支持预算项新增、编辑、删除
- 支持录入“已使用额度”
- 全流程在同一页面通过底部弹层完成（无页面跳转）
- 当前阶段基于本地存储实现数据持久化（无服务端）

## 2. 入口与页面结构

### 2.1 入口

- 首页工具入口：`pages/index/index.js`
  - 工具名：`预算看板`
  - 跳转路径：`/pages/budget/budget`

### 2.2 页面

- 主页面：`pages/budget/budget`
  - 展示月度预算总览与预算项列表
  - 负责全部交互（新增、编辑、删除、设置已使用、滑动操作、弹层动画、振动反馈）

## 3. 功能说明

### 3.1 月度视图（自然月）

- 默认展示当前自然月（格式：`YYYY-MM`）
- 可左右切换月份
- 每个月份的数据彼此独立存储

### 3.2 预算列表展示

每个预算项显示：

- 预算项名称
- 已用金额 / 预算总额
- 进度条（马卡龙配色轮换）
- 剩余金额（超支时高亮提示）

### 3.3 新增预算项

- 点击右下角 `+` 打开底部弹层
- 输入预算项名称
- 使用假键盘输入预算金额
- 保存后写入本地并刷新列表

### 3.4 编辑预算项（右滑）

- 预算项右滑后露出左侧 `编辑`
- 点击 `编辑` 打开底部弹层（复用“新增预算项”弹层）
- 可修改名称与预算金额（金额同样使用假键盘）

### 3.5 删除预算项（左滑）

- 预算项左滑后露出右侧 `删除`
- 点击 `删除` 后二次确认
- 确认后从本地存储移除并刷新列表

### 3.6 设置已使用额度（点击预算项）

- 点击预算项卡片打开“设置已使用额度”弹层
- 使用计算器样式假键盘输入（数字、`.`、`00`、退格、清空、保存）
- 保存后刷新进度与汇总

## 4. 交互与动效设计

### 4.1 弹层

- 从底部弹出，含遮罩层
- 打开：遮罩淡入 + 面板上滑渐显
- 关闭：遮罩淡出 + 面板下滑渐隐
- 通过“状态位 + 延迟卸载”实现关闭动画

### 4.2 假键盘反馈

- 按下时有视觉反馈（缩放、颜色/亮度、内阴影）
- 按键触发轻量振动反馈（`wx.vibrateShort()`）

### 4.3 滑动操作

- 支持左右滑动并吸附到操作位
- 点击已滑开卡片会先收起，避免误触
- 切换月份时会重置滑动状态

## 5. 数据与存储设计

存储工具文件：`utils/budget-storage.js`

### 5.1 Storage Key

- `budget_items_v1`

### 5.2 月度分桶结构

```json
{
  "2026-03": [
    {
      "id": "b_1740000000000_1234",
      "name": "餐饮",
      "totalAmount": 1500,
      "usedAmount": 520,
      "createdAt": 1740000000000
    }
  ]
}
```

### 5.3 主要方法

- `getCurrentMonthKey(date?)`
- `getMonthBudgetList(monthKey)`
- `addBudgetItem(monthKey, payload)`
- `updateBudgetItem(monthKey, id, payload)`
- `updateUsedAmount(monthKey, id, usedAmount)`
- `deleteBudgetItem(monthKey, id)`
- `getBudgetItem(monthKey, id)`

## 6. 关键实现点（budget 页面）

主逻辑文件：`pages/budget/budget.js`

- 列表计算：金额格式化、进度百分比、超支判断、汇总统计
- 弹层状态：新增/编辑弹层、已使用弹层的显示与关闭动画状态
- 假键盘输入：统一金额规整逻辑（最多 2 位小数）
- 手势滑动：记录 touch 起点并计算横向位移，触发操作区露出
- 触感反馈：关键按键动作触发短振动

## 7. 维护建议

- 若后续接入服务端，可将 `budget-storage.js` 替换为 API 层，同时保留本地缓存能力
- 可将假键盘区域抽成自定义组件，减少 `budget.wxml` 重复结构
- 可增加“预算项排序”“分类图表”“导出月报”等扩展功能

## 8. 手动测试清单

- 进入预算页：默认显示当前月
- 新增预算：名称+金额保存成功，列表刷新
- 编辑预算：右滑编辑，修改后生效
- 删除预算：左滑删除，确认后移除
- 设置已使用额度：点击项进入弹层，保存后进度与汇总更新
- 弹层开关动画：进入/退出均顺滑
- 假键盘反馈：视觉反馈 + 轻振动正常
- 切月：各月数据隔离，显示正确
